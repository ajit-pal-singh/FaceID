Face Recognition Project

This is a Face Recognition Project developed to automatically identify individuals based on facial features. It leverages machine learning algorithms and computer vision techniques to process and analyze faces captured through a webcam or uploaded images. Upon successful identification, it displays details such as the user's name, age, and gender.

Technologies Used:

  • Frontend: HTML, CSS, JavaScript
  • Backend: Python (Flask)
  • Face Recognition: OpenCV, dlib, face_recognition library
  • Animation: Lottie for smooth animations

User Flow:

  • The user clicks on the "Start Recognition" button or uploads a photo.
  • The system captures the face and sends it to the backend for recognition.
  • If a match is found, the user's details (name, age, gender) are shown in a pop-up card.
  • If no match is found, a relevant error message is displayed.
